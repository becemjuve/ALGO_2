	1/ Start
	2/ Input the vectors v1 and v2 of length n
	3/ Initialize the variable ps to 0
	4/ For i from 1 to n, do steps 5-6
	5/ Multiply the i element of v1 with the i element of v2
	6/ Add the result to ps
	7/ Return ps
	8/ End

	2/ Input the number of pairs of vectors n
	3/ For i from 1 to n, do steps 4-5
	4/ Input the vectors v1 and v2 of length m
